How many Newton steps does it take to optimize the intercept?

Consider the case where \(y \sim \text{Bernoulli}(\frac{1}{1 + \exp\{b_0 + \psi_i\}})\) for known \(\psi_i\). How do we optimize the intercept \(b_0\)?

Write \(\mu_i = \frac{1}{1 + \exp\{b_0 + \psi_i\}}\).

\[ \begin{align} L(b_0) &= \sum y_i(b_0 + \psi_i) - \log(1 + \exp \{ b_0 + \psi_i \}) \\ \frac{d}{db_0} L &= \sum y_i - \mu_i \\ \frac{d^2}{db_0} &= -\sum \mu_i(1 - \mu_i) \end{align} \]

optimize_b0 <- function(y, psi, b0_init = 0, maxit=10, tol=1e-10){
  b0 <- b0_init
  for (i in 1:maxit){
    mu <- 1 / (1 + exp(-(b0 + psi)))
    grad <- sum(y - mu)
    hess <- -sum(mu *(1 - mu))
    dir <- grad/hess
    b0 <- b0 - dir
    nd <- -0.5 * grad^2/hess
    if(nd < tol){
      break
    }
  }
  return(list(b0 = b0, iter=i, nd = nd, grad=grad, hess = hess))
}
